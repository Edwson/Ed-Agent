What this is — and what it isn’t

  • It is a window onto real runs that already happened — parsed from the human-readable memory file, the single source of truth.
  • It edits the same file by hand-editable lines, so power users can still use git and a text editor.
  • It runs on 127.0.0.1 only and makes zero external requests — your runs never leave the machine.
  • It is not a live always-on daemon — Ed Agent runs are discrete, so this is a control room, not a real-time monitor.
  • Token figures are estimates (~4 chars/token) for cost governance — not a bill.
  • The engine is the CLI / MCP. This is a companion surface; it observes and tunes, it doesn’t replace them.
